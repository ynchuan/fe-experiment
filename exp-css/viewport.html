小结： 1、sreen宽度与devicePixelRatio设备像素比的乘积为手机物理像素 2、设备宽度（device-width）: 这个就是DIP的值，即是设备独立像素，由设备硬件本身决定；device-width一般值范围是320—432（iPhone4S 是 320， iPhone6 plus 是 414， 魅蓝 是 432） 3、可视区宽度（layout viewport） = 设备独立像素(visual viewport) / 缩放比；可视区宽度与viewport width比较大小决定是否出现滚动条 4、滚动条出现规则： 可视区宽度<内容宽度，出现滚动条； 可视区宽度>内容宽度,内容宽度=可视区宽度，不出现滚动条
